ExponentialBackoffRetry

class ExponentialBackoffRetry(val maxAttempts: Int = 3, initialDelayMs: Long = 1000, maxDelayMs: Long = 30000, factor: Double = 2.0) : RetryPolicy

Exponential backoff retry policy.

Delays grow exponentially: initialDelayMs * (2 ^ attempt)

Example delays with initialDelayMs=1000:

  • Attempt 1: 1s

  • Attempt 2: 2s

  • Attempt 3: 4s

  • Attempt 4: 8s (capped at maxDelayMs)
