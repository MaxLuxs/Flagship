ServerAnalytics

class ServerAnalytics(projectId: String, analyticsUrl: String, apiKey: String? = null, batchSize: Int = 10, flushIntervalMs: Long, maxRetries: Int = 3, logger: FlagsLogger = NoopLogger) : FlagsAnalytics

Analytics implementation that sends events to Flagship server.

Automatically batches events and sends them periodically to reduce network overhead.

Example:

val analytics = ServerAnalytics(
    projectId = "project-id",
    analyticsUrl = "https://api.flagship.io",
    apiKey = "your-api-key"
)

val config = FlagsConfig(
    appKey = "my-app",
    analytics = analytics,
    // ...
)
