CircuitBreaker

class CircuitBreaker(failureThreshold: Int = 5, successThreshold: Int = 2, timeoutMs: Long, clock: Clock = SystemClock)

Circuit breaker for protecting providers from cascading failures.

When failures exceed threshold, circuit opens and rejects requests. After timeout, circuit enters half-open state to test recovery.
