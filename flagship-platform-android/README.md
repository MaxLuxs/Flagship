# Flagship Platform Android

Android-specific implementations for **Flagship**.

## Features

- **PersistentCache**: Uses `SharedPreferences` to store flag snapshots offline.
- **Context Helpers**: Utilities to extract `EvalContext` from Android `Context` (version, OS, device ID).

## Installation

```kotlin
implementation("io.maxluxs.flagship:flagship-platform-android:1.0.0")
```
